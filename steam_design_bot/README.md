# 🎨 Steam Design Bot

Telegram-бот для приёма заявок на индивидуальное оформление профилей Steam.

---

## ⚡ Быстрый старт

### 1. Установи зависимости
```bash
pip install -r requirements.txt
```

### 2. Настрой config.py

Открой `config.py` и заполни три поля:

```python
BOT_TOKEN = "ВСТАВЬ_ТОКЕН_СЮДА"    # Токен от @BotFather
ADMIN_ID = 123456789                # Твой Telegram ID (узнай у @userinfobot)
ADMIN_USERNAME = "ryzyni"           # Твой username без @
```

### 3. Добавь фото-примеры

Создай папку `media/` рядом с `bot.py` и положи туда 5 картинок:

| Файл | Что это |
|------|---------|
| `illus_vertical.jpg` | Пример вертикальной иллюстрации |
| `illus_horizontal.jpg` | Пример горизонтальной иллюстрации |
| `bg_animated.jpg` | Скриншот/превью анимированного фона |
| `bg_static.jpg` | Пример обычного фона |
| `portfolio_1.jpg` | Пример готовой работы |

### 4. Запусти бота
```bash
python bot.py
```

## 🔄 Сценарий работы бота

```
/start
  └─ Главное меню (3 кнопки)
       ├─ 📋 Условия — объяснение услуги, требования, цены
       ├─ 💬 Примеры работ — фото из портфолио
       └─ 🎨 Заказать оформление
            └─ Шаг 1: Вертикальная / Горизонтальная иллюстрация
                 └─ Показывает пример фото + [Продолжить / Назад]
                      └─ Шаг 2: Анимированный / Обычный фон
                           └─ Показывает пример фото + [Продолжить / Назад]
                                └─ Шаг 3: Итоговый счёт (250 руб / 200 ⭐)
                                     └─ [Согласен] → Заявка отправляется тебе
                                          └─ Ты видишь уведомление с кнопками:
                                               ├─ ✅ Одобрить → клиент получает твой @
                                               └─ ❌ Отклонить → клиент получает отказ
```
